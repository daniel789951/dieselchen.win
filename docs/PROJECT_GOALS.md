# 專案規格與規劃書 (Project Specifications)

## 1. 專案目的 (Project Purpose)

**「Dieselchen.win - 可以回家啦」** 是一個專為辛勤上班族打造的療癒系網頁應用程式。
其核心目的在於：

* **心理慰藉**：透過精準的下班倒數計時，讓使用者看到「希望」在接近。
* **幽默紓壓**：結合時下流行的「躺平文化」與職場梗，用幽默化解工作的枯燥。
* **儀式感**：在午休與下班時刻提供專屬的提醒與慶祝動畫，建立一種「終於結束了」的儀式感。

## 2. 現有功能 (Current Features)

### 2.1 核心功能

* **多時區/班次倒數**：
  * 支援「正常班 (17:30)」、「晚班 (18:30)」、「金控 (18:00)」三種下班時間倒數。
  * 精確計算時、分、秒，即時更新。
* **情境式視覺回饋**：
  * **上班時間**：顯示「疲憊上班族」GIF，搭配各式職場厭世語錄。
  * **下班時間**：顯示「開心回家」GIF，搭配下班慶祝語錄與背景動畫。

### 2.2 互動提醒

* **午休充電站**：每日 12:00 - 13:30 跳出溫馨的午休提醒彈窗，提醒使用者暫時放下工作。
* **下班慶祝儀式**：平日 17:30 後（至隔日 07:00），首次進入或時間到達時，觸發慶祝彈窗與「叮」的提示音效，並伴隨五彩碎紙與氣球動畫。

### 2.3 趣味擴充

* **每日躺平激勵**：右下角常駐小工具，點擊可隨機獲取一句結合時事哽的「毒雞湯」或「躺平語錄」，博君一笑。

## 3. 建議發展方向 (Future Roadmap)

為了增加用戶黏著度與趣味性，未來可考慮以下發展方向：

### 3.1 個人化設定 (Personalization) - *高優先*

* **自訂下班時間**：允許使用者輸入自己的上下班時間，網頁透過 LocalStorage 記憶設定，不再侷限於預設的三種班次。
* **自訂稱呼**：讓使用者設定自己的名字或職稱（e.g. "辛苦了，陳經理"）。

### 3.2 社交互動 (Social Features)

* **下班打卡分享**：一鍵生成「我下班了！」的圖片卡片，包含當下的倒數為 0 的畫面，方便分享至 Instagram 限時動態或 LINE 群組。
* **比慘大會**：簡單的投票或留言功能，比較誰加班最晚（需評估後端成本）。

### 3.3 功能增強 (Enhancements)

* **PWA 支援**：將網站設定為漸進式網頁應用 (PWA)，讓手機用戶可以將其「安裝」在桌面，像 App 一樣開啟，並支援離線倒數。
* **更多音效包**：除了「叮」聲，可增加如「下課鐘聲」、「瑪利歐過關」等音效供選擇。
* **節日彩蛋**：遇特定節日（如過年、聖誕節）自動更換背景主題或特殊語錄。

### 3.4 視覺優化

* **深色/淺色模式切換**：目前預設偏暗色系，可增加明亮模式切換。
