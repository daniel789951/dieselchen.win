# 使用說明手冊 (User Manual)

## 歡迎來到 DieselChen.win

這是一個為上班族打造的下班倒數計時與激勵網站。以下是您可以在這裡體驗的功能：

### 1. 核心體驗

#### 班次倒數計時
網站左上角顯示兩種班次的下班倒數：
- **正常班**: 8:50 - 17:30
- **金控班**: 9:00 - 18:00

#### 智能狀態切換
網站會根據時間自動切換不同的視覺狀態：
- **上班中**: 顯示疲憊上班族 GIF 與勵志（厭世）語錄
- **午休時間** (11:50-13:30): 顯示吃午餐 GIF，語錄自動隱藏
- **下班後**: 顯示開心回家 GIF 與下班慶祝語錄

#### 每日躺平激勵
右下角隨機顯示結合時事與「躺平文化」的幽默勵志語錄，點擊即可切換。

#### 技術顧問連結
右上角提供兩位技術顧問的部落格連結：
- **安顧問**: 技術分享與開發心得
- **達顧問**: 技術支援與問題解決

### 2. 貼心提醒功能

#### 午休提醒彈窗
- **時間**: 每日 12:00 - 13:30
- **功能**: 自動跳出休息提示，可手動關閉
- **特色**: 同一時段內關閉後不會重複打擾

#### 下班慶祝彈窗
- **時間**: 平日 17:30 - 隔日 07:00
- **功能**: 下班慶祝視窗，含音效與五彩碎紙、氣球動畫
- *注意*: 音效可能會很震撼，辦公室瀏覽請注意音量

#### 打卡提醒推播 (NEW!)
- **時間**: 每日 17:30
- **功能**: 發送桌面通知，提醒記得打卡下班
- **設定**: 首次使用需授予瀏覽器通知權限
- **提示**: 即使網頁在背景執行，也能收到提醒

### 3. 小遊戲中心 (Mini Games)

位於畫面左下角的 "G" 按鈕是您的舒壓中心：

- **拉霸機 (Slots)**: 試試您的手氣，看能不能轉出三個 Diesel！
- **什麼時候吃飯/喝什麼 (Roulette)**: 有選擇障礙？讓命運之輪幫您決定午餐或下午茶。
- **射飛鏢 (Darts)**: 簡單的舒壓小遊戲，點擊螢幕發射！

### 常見問題 (FAQ)

**Q: 為什麼會有聲音？**
A: 下班時刻會自動播放慶祝音效，您可以通過瀏覽器分頁靜音，但建議佩戴耳機體驗最佳效果。

**Q: 如何啟用桌面通知？**
A: 首次訪問網站時，瀏覽器會詢問是否允許通知。請點擊「允許」即可啟用。如果不小心拒絕了，可以在瀏覽器設定中重新開啟網站的通知權限。

**Q: 為什麼午休時間看不到語錄？**
A: 這是特意設計的功能。在 11:50-13:30 午休時間，語錄會自動隱藏，讓畫面更簡潔，專注於休息。

**Q: 晚班的倒數計時去哪了？**
A: 為了簡化介面，目前僅保留「正常班」和「金控班」兩種班次的倒數計時。

**Q: 畫面跑版了怎麼辦？**
A: 請嘗試重新整理頁面，或確認您的瀏覽器是否為最新版本。本站支援響應式設計 (RWD)。
